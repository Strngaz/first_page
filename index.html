So lets begin

Programming refers to a technological process for telling a computer which tasks to perform in order to solve problems. You can think of programming as a collaboration between humans and computers, in which humans create instructions for a computer to follow (code) in a language computers can understand. 

Just as recipes have their own syntax, so the programs have their own syntax. Every program is written in some programming language that has its own syntax. Every program is translated into machine code which machine (computer, tablet, phone) executes.

The program expects input data and instructions for working with data. A program always has some form of data for processing. Data can be hardcoded or data which depend on external factors (input user, bar code reading, scanning documents etc.)

Program development is simply called programming. Program development implies use special tools - programming languages.
